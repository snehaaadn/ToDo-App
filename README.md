# ToDo-App

This is a simple yet efficient ToDo application built using Swift and SwiftUI. It allows users to create, manage, and delete daily tasks with a smooth and intuitive interface. The app leverages local storage to persist data across sessions without the need for an internet connection.

## Features:

- Add new tasks with a clean UI
- Mark tasks as completed
- Delete tasks
- Persistent data storage using CoreData
- Built using SwiftUI for a modern and responsive design

## Technologies Used:

- Swift 5
- SwiftUI
- Local storage (CoreData)
- MVVM architecture

## Requirements

- iOS 14.0+
- Xcode 12.0+
- Swift 5


## Installation

1. Clone this repository:

https://github.com/snehaaadn/ToDo-App

3. Open the project in Xcode.

4. Build and run the app on a simulator or physical device.

## Usage

1. Launch the app on your iOS device or simulator.
2. Sign in by just entering name and password.
3. Create a new task by tapping the "Add Task" or + button.
4. Click on the circle or double tap on tasks to complete them.
5. Slide left a tast or press on edit to delete them.


## Architecture

1. MVVM (Model-View-ViewModel) architecture.
2. Core Data/Realm for data persistence.
3. SwiftUI for UI or Swift for backward compatibility.
